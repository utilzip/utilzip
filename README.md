# utilzip

간소화된 유틸리티 라이브러리

## 특징

- 배열, 객체, 문자열을 다루기 위한 사용하기 쉬운 API 제공
- 외부 의존성 없이 경량화된 라이브러리

## 설치

```
npm install utilzip
```

## 사용법

```js
const utilzip = require("utilzip");
```
